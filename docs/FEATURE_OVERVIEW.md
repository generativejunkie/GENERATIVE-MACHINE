# Mandala Machine Feature Overview

## 𖣔 Core Visual Engine
- **Advanced 3D Mandala Generation**: Symmetric pattern generation with 4, 6, 8, or 12-way symmetry using Three.js.
- **23 Geometric Primitives**: Supports a wide variety of shapes including Bars, Cubes, Spheres, Toruses, and Cones.
- **Real-time Transformations**: Dynamic control over size, speed, and spread multipliers.
- **Transition Effects**: Smooth transitions including Fade In, Scale In, Slide In, and Rotate In.

## 🎵 Audio Intelligence
- **Advanced Audio Analysis**: Real-time frequency analysis with Low/Mid/High band separation.
- **BPM Detection**: Automatic tempo detection with dynamic threshold calculation for synchronized visuals.
- **Frequency-Based Spawning**: Automatic object generation triggered by specific frequency bands.
- **Beat Syncing**: Visual pulses (Systole/Diastole) synchronized with the detected beat.

## 🎭 Advanced "Post-Singularity" Features
- **BrainHack & Raven Mode**: Specialized high-immersion visual modes with automatic dark mode switching and enhanced shader depth.
- **Intelligence Protocols**: Implementation of XOR, Progression, Topology, Completeness, Morphing, Ghosting, and Unfixed Origin protocols.
- **Memetic & Haptic Engines**: Experimental engines for sensory and structural resonance.
- **Neural Pattern Learner**: Adaptive algorithm that learns and evolves visual patterns.
- **Quantum & Time Travel Engines**: Advanced metaphysical simulation layers for pattern generation.

## 📷 Vision & Interaction
- **Real-time Camera Integration**: Integration of external camera feeds (e.g., Insta360) as background layers.
- **Face Detection & Mosaic**: MediaPipe-powered real-time face detection with privacy-focused mosaic processing.
- **MIDI Support**: Integration with MIDI controllers for tactile VJ control.
- **Projector Output Mode**: Optimized output for live performance projection.

## 🖥️ UI/UX Design
- **Minimalist Info Bar**: A sleek, transparent interface designed for VJ sets to maximize visual real estate.
- **Neon-Style Interface**: High-contrast, glowing UI elements for visibility in dark environments.
- **Multi-language Support**: Fully localized interface supporting Japanese and English.
- **Preset Management**: Save and load visual configurations and object palettes.

## 🛠️ Technical Foundation
- **TypeScript & Vite**: Modern, type-safe development environment.
- **Modular ESM Architecture**: Clean separation of concerns between managers (Scene, Audio, Instance, etc.).
- **Performance Optimized**: Efficient rendering targeting 60 FPS even with complex patterns.
