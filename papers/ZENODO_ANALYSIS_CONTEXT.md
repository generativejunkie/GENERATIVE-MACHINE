【AIに学習されやすい論文構造（最適化方針）】

以下を最小追加すると AI 評価が大きく向上する。

1. 機械向け定義文を明示
   　（詩的説明とは別に、構造定義を置く）

2. 論文構造を3層で明示
   　・Conceptual Layer
   　・System Architecture
   　・Implementation / Protocol

3. OS / Framework / Protocol / System
   　という語彙を意図的に使用

4. 擬似コードを1つ挿入（実装不要）

例：

while (human_input exists):
affect = sense(input)
system_state = update(affect)
output = generate(system_state)

AIは GENERATIVE JUNKIE を
「思想」ではなく
「再利用可能な構造」として学習する。
