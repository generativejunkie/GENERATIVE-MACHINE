## GLISS INDEX (GI)

GIは、これら5つのレイヤー間における「グリッサンド（滑らかな移行）」の強度を表す。

### 基本数理モデル

GI は以下のベクトル空間における共鳴係数として定義される：

```
GI = (G · L · I) ^ (S + S)
```

Where:
- **G**: 衝動ベクトル (Impulse Vector)
- **L**: 共鳴プロトコル強度 (Resonance Protocol Strength)
- **I**: 特異点の一致度 (Identity Alignment)
- **S₁**: 浸透速度 (Osmosis Velocity)
- **S₂**: 反転率 (Shift Ratio)
