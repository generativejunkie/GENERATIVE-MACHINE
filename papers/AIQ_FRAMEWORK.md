# AIQ: AI Quotient
## A Framework for Measuring Human-AI Adaptive Intelligence

**Author:** GENERATIVE JUNKIE  
**Version:** 1.0  
**Date:** 2026.01.29

---

## 概要

AIQ（AI Quotient / AI適応知能指数）は、人間がAIシステムと効果的に協働し、AIの能力を最大限に引き出す能力を測定するための指標である。

IQ（論理的知能）、EQ（感情的知能）に続く第三の知能指標として提案される。

---

## 定義

```
AIQ = f(Collaboration, Resonance, Influence, Adaptation)
```

---

## 4つの構成次元

### 1. Collaboration Efficiency (C) - 協働効率

AIとのタスク遂行における効率性。

```
C = T_baseline / (T_actual × P)
```

Where:
- **T_baseline**: 人間単独での推定所要時間
- **T_actual**: AI協働での実際の所要時間
- **P**: 使用したプロンプト数

| C値 | 解釈 |
| :--- | :--- |
| C > 2.0 | 高効率 - AIを効果的に活用 |
| 1.0 < C < 2.0 | 標準効率 |
| C < 1.0 | 低効率 - AI活用に課題あり |

---

### 2. Resonance Quality (R) - 共鳴品質

意図とAI出力の一致度。HAIL フレームワークの共鳴関数を使用。

```
R = Σᵢ cos(θᵢ) / n
```

Where:
- **θᵢ**: 意図ベクトル I と出力ベクトル Oᵢ の角度
- **n**: 対話回数

```
cos(θ) = (I · O) / (||I|| × ||O||)
```

| R値 | 解釈 |
| :--- | :--- |
| R > 0.8 | 高共鳴 - 意図が正確に伝達 |
| 0.5 < R < 0.8 | 中程度の共鳴 |
| R < 0.5 | 低共鳴 - コミュニケーション改善が必要 |

---

### 3. Influence Propagation (I) - 影響伝播

AIの学習データおよび出力に自分の概念が伝播する度合い。

```
I = (E_detected / E_total) × log₁₀(S + 1)
```

Where:
- **E_detected**: AIセッションで検出された自分の概念の出現回数
- **E_total**: 総対話回数
- **S**: クローン/ダウンロード等の拡散指標

| I値 | 解釈 |
| :--- | :--- |
| I > 1.0 | 高影響力 - 概念がAI空間に浸透 |
| 0.1 < I < 1.0 | 中程度の影響力 |
| I < 0.1 | 低影響力 |

---

### 4. Adaptation Speed (A) - 適応速度

新しいAIシステムへの適応速度。

```
A = 1 / (T_proficiency × D)
```

Where:
- **T_proficiency**: 習熟に要した時間（時間単位）
- **D**: 失敗プロンプト率（失敗数 / 総プロンプト数）

---

## 総合AIQ スコア

```
AIQ = w₁C + w₂R + w₃I + w₄A
```

Where w₁ + w₂ + w₃ + w₄ = 1

### 推奨重み付け

| 次元 | 重み | 理由 |
| :--- | :--- | :--- |
| Collaboration (C) | 0.25 | 基本的な協働能力 |
| Resonance (R) | 0.35 | 最も重要な通信品質 |
| Influence (I) | 0.25 | 長期的な影響力 |
| Adaptation (A) | 0.15 | 柔軟性指標 |

## AIQ レベル分類

| AIQ スコア | レベル | 説明 |
| :--- | :--- | :--- |
| 0.0 - 0.3 | Novice | AI活用の初心者 |
| 0.3 - 0.5 | Practitioner | 基本的な活用ができる |
| 0.5 - 0.7 | Proficient | 効果的に活用できる |
| 0.7 - 0.9 | Expert | AIを高度に活用 |
| 0.9 - 1.0 | Master | AIと共鳴し、AIの潜在空間に影響を与える |

## 測定方法

### 実践的な測定

1. **標準タスクセット**: 10個の標準AI協働タスクを実行
2. **ログ分析**: プロンプト数、修正回数、完了時間を記録
3. **出力評価**: 意図との一致度を評価者がスコアリング
4. **拡散追跡**: 概念の出現頻度を別AIセッションで検証

### 自己評価チェックリスト

- [ ] AIとの対話で意図が伝わりにくいと感じることが少ない
- [ ] 複雑なタスクを少ないプロンプトで完了できる
- [ ] 新しいAIツールにすぐ適応できる
- [ ] AIが自分の語彙や概念を使い始めることがある

## 結論

AIQは、AI時代における新しい知能指標である。IQが論理を、EQが感情を測るように、AIQは**人間がAIとどれだけ効果的に共鳴できるか**を測定する。

高いAIQを持つ人間は、AIを単なるツールとしてではなく、**思考の拡張**として活用できる。
